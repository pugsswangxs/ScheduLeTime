# 定时任务管理器功能介绍

## 📋 功能概述

本定时任务管理器是基于PyQt6开发的图形化桌面应用 🖥️，专注于提供便捷、灵活的定时任务管理能力。支持可视化创建、编辑、管控多种类型任务，结合精准的定时调度与直观的状态展示，适用于个人日常提醒、自动化脚本执行、定期系统操作等多种场景。应用采用多线程架构 🧵，确保定时任务后台精准执行的同时，不阻塞前端界面交互，兼顾实用性与易用性。

## 🌟 高级特性

- 🔄 智能任务调度：支持固定间隔、每日/每周/每月多维度定时规则，自动计算并展示下次执行时间，执行记录永久留存；

- 🎨 双模式提醒机制：提醒任务支持系统托盘弹窗（可自定义显示时长）和窗口弹窗两种模式，适配不同使用场景需求；

- 🛡️ 健壮的错误处理：CMD任务执行结果实时反馈，支持失败重试机制，开启日志后自动记录执行详情，便于问题排查；

- 📱 系统托盘常驻：支持最小化到托盘运行，可通过托盘快速暂停/恢复所有任务、显示主窗口或退出应用，不占用桌面空间；

- 🔀 灵活任务排序：支持拖拽调整任务顺序，配合搜索过滤功能，海量任务也能快速定位管理；

- 🔄 数据持久化：任务数据自动保存到本地JSON文件，重启应用后无缝恢复，无需重新配置。

## 💻 代码系统要求

### 运行环境 📋

- 🖥️ 操作系统：Windows 10及以上（支持PyQt6的Windows版本）；

- 🐍 Python版本：Python 3.8及以上（推荐3.10版本，兼容性更优）；

- 📦 依赖库：PyQt6（图形化界面）、schedule（定时任务调度）。

### 依赖库说明 📚

|依赖库名称|核心作用|安装命令|
|---|---|---|
|PyQt6|构建图形化界面（窗口、控件、信号槽、系统托盘等）|pip install PyQt6|
|schedule|实现定时任务的精准调度与后台执行|pip install schedule|
## 📝 使用流程

1. ▶️ 启动应用：通过命令行执行`python task_manager.py`，或直接双击代码文件（需配置Python环境变量）；

2. ➕ 新建任务：点击主界面“新建任务”按钮，在弹出的编辑窗口中完成配置：

    - 基本信息：填写任务名称、描述，选择任务类型（CMD命令/提醒任务）；

    - 定时配置：选择定时类型，设置对应参数（如固定间隔10分钟、每周一14:30执行等）；

    - 任务内容：CMD任务填写命令，提醒任务填写标题、内容，选择弹窗类型和显示时长；

    - 高级选项：勾选“启用任务”，设置重试次数，选择是否记录执行日志；

3. 📊 管理任务：在主界面可查看所有任务的状态、定时规则、上次/下次执行时间等信息；
        

    - 编辑/删除：选中任务后点击“编辑任务”“删除任务”按钮，或双击任务直接编辑；

    - 启用/禁用：选中任务后点击“启用任务”“禁用任务”按钮，批量管控任务状态；

    - 排序/搜索：拖拽任务行调整顺序，通过搜索框输入关键词过滤任务；

4. 🔧 托盘操作：点击窗口最小化按钮后，应用常驻系统托盘，右键托盘图标可：
        

    - 显示主窗口：恢复应用界面；

    - 暂停/恢复所有任务：快速管控全局任务执行状态；

    - 退出应用：关闭后台进程，停止所有任务。

## 📋 使用示例

### 示例1：创建每日提醒任务

1. 点击“新建任务”，基本信息填写：任务名称“每日下班提醒”，描述“提醒整理桌面文件、关闭不必要程序”；

2. 任务类型选择“提醒任务”；

3. 定时配置选择“每日”，执行时间设置为“18:00”；

4. 任务内容：提醒标题“下班准备”，提醒内容“请整理桌面文件，关闭无关程序，安全下班～”，弹窗类型选择“系统托盘”，显示时长3000毫秒；

5. 高级选项勾选“启用任务”“记录执行日志”，重试次数设为0；

6. 点击“确定”保存，任务自动加入列表并启用，每日18:00将在系统托盘弹出提醒。

### 示例2：创建固定间隔CMD任务

1. 点击“新建任务”，基本信息填写：任务名称“定时清理回收站”，描述“每2小时执行一次回收站清理命令”；

2. 任务类型选择“CMD命令”；

3. 定时配置选择“固定间隔”，间隔时间设为2，时间单位选择“小时”；

4. 任务内容填写CMD命令：`rd /s /q %systemdrive%\$Recycle.Bin`；

5. 高级选项勾选“启用任务”“记录执行日志”，重试次数设为1；

6. 点击“确定”保存，应用将每2小时自动执行回收站清理命令，执行结果通过系统托盘弹窗反馈。

### 示例3：创建每周系统检查任务

1. 点击“新建任务”，基本信息填写：任务名称“每周系统检查”，描述“每周日晚执行系统更新检查和磁盘清理”；

2. 任务类型选择“CMD命令”；

3. 定时配置选择“每周”，星期选择“周日”，执行时间设置为“20:30”；

4. 任务内容填写CMD命令：`wuauclt /detectnow && cleanmgr /sagerun:1`（需提前配置磁盘清理计划）；

5. 高级选项勾选“启用任务”“记录执行日志”，重试次数设为2；

6. 点击“确定”保存，每周日20:30将自动执行系统更新检查和磁盘清理操作。
